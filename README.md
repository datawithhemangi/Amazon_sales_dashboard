# Amazon_sales_dashboard
Amazon Sales Dashboard using Microsoft Excel

Interactive Amazon Sales Dashboard using Microsoft Excel to analyze sales performance and extract meaningful business insights.

The dashboard focuses on answering key business questions:

# 🔹 Regional Sales Performance
 ➡️ Which regions are contributing the highest sales, and how does sales performance vary across different regions?

# 🔹 Product Performance Analysis
 ➡️ Which products are generating the highest sales, and which products need more attention?

# 🔹 Payment Method Analysis
 ➡️ Which payment methods are preferred most by customers, and what is the distribution across different payment options?

# 🔹 Delivery Status Insights
 ➡️ What is the overall distribution of orders based on delivery status (Delivered, In Transit, Cancelled)?

Using Excel features like:

✅ Pivot Tables

✅ Pivot Charts

✅ Slicers

✅ Data Visualization

✅ Dashboard Design
